# Organização LP-CEFETMG

Bem-vindo à organização da disciplina **Linguagem de Programação** do  
**Departamento de Computação (DECOM) - CEFET-MG**.

## 🎓 Sobre a disciplina
A disciplina de Linguagem de Programação é oferecida nos cursos técnicos de:
- **Informática**
- **Redes de Computadores**

Nosso objetivo é desenvolver competências em programação estruturada e orientada a objetos, preparando os alunos para desafios acadêmicos e profissionais na área de tecnologia.

## 📂 Repositório principal
O repositório oficial da disciplina está disponível em:  
👉 [LP_CEFETMG](https://github.com/lp-cefetmg/LP_CEFETMG)

Nesse repositório você encontrará:
- Materiais de apoio
- Exercícios e projetos
- Exemplos de código
- Orientações para submissão de atividades

## 🚀 Como participar
- Acesse os repositórios fixados para acompanhar os projetos da disciplina.
- Contribua com commits e pull requests conforme as instruções dos professores.
- Utilize o GitHub como ferramenta de aprendizado e colaboração.
